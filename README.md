# PonyLib v2

Fb2 library with web interface.

**Work in progress**

## Tools

### `ponylib-parser`

Parse fb2 files & display metadata. Can be used separetly.

Support recursive directory scan, follow symlinks.

Usage:

```
ponylib-parser FILE_OR_DIR [FILE_OR_DIR]
```
